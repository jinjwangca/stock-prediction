# SOFI Stock Price Prediction

This project predicts the next-day closing price of SoFi (SOFI) using machine learning.

## ML Models
- Linear Regression
- Random Forest
- XGBoost

## Files
- `notebooks/`: Jupyter notebook
- `report/`: HTML version
- `images/`: Model result plots
- `data/`: Cleaned features
